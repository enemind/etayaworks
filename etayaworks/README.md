## 🚀 Deploy to GitHub Pages (Free Hosting)

### Step 1 — Create a GitHub Account
Go to https://github.com and sign up (free).

### Step 2 — Create a New Repository
1. Click the **+** icon → **New repository**
2. Name it: `etayaworks` (or anything you like)
3. Set to **Public**
4. Click **Create repository**

### Step 3 — Upload Your Files
**Option A — Drag & Drop (easiest):**
1. In your new repo, click **uploading an existing file**
2. Drag all files and folders (`index.html`, `css/`, `js/`) into the upload area
3. Click **Commit changes**

**Option B — Git (if you have Git installed):**
```bash
git init
git add .
git commit -m "Initial commit – Etayaworks website"
git branch -M main
git remote add origin https://github.com/YOUR-USERNAME/etayaworks.git
git push -u origin main
```

### Step 4 — Enable GitHub Pages
1. Go to your repo → **Settings** → **Pages** (left sidebar)
2. Under **Source**, select **Deploy from a branch**
3. Choose **main** branch → **/ (root)**
4. Click **Save**
5. Wait ~2 minutes, then visit: `https://YOUR-USERNAME.github.io/etayaworks/`
